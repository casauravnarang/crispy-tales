# Crispy Tales 🍿✨

A premium D2C makhana brand website, built with React and Tailwind CSS.

## Folder Structure

```
public/
src/
  ├── components/
  ├── pages/
  ├── styles/
```

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
